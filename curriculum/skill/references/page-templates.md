# Page Templates Reference

This file contains the exact structural templates for every worksheet page type.
These patterns were extracted from Ashley Larkin's proof-of-concept packets
(Charlie's 1st grade and Noelle's 3rd grade) and represent the quality bar.

## How to Use

When generating a specific page type, find the matching template below.
Follow the structure exactly. The content changes weekly based on the scope
and sequence, but the FORMAT stays consistent. Parents should recognize
the layout from week to week.

## MATH PAGE TEMPLATES

### Template M1: Computation Practice (All Grades)

**Used for:** Addition, subtraction, multiplication, division fact practice

**Structure:**
```
[Title: e.g., "Multiplication Practice"]
[TEKS reference]
[Name/Date lines]

Section 1: "Speed Round! Solve as fast as you can:" (or similar energizing header)
- 20 problems in a 4-column x 5-row grid
- Pure computation, no word problems
- Progresses from easier to harder across the grid
- Format: "1. 3 x 4 =    2. 7 x 6 =    3. 8 x 5 =    4. 9 x 3 ="

Section 2: "Solve these (show your work in the box):" (6 problems)
- Multi-digit or more complex versions of the same operation
- Vertical format with work space boxes next to each problem
- Format:
  "  34        27        48
   x  5       x  3       x  6
  [work]     [work]     [work]"

[Footer tip about the operation, e.g., "Multiplication is repeated addition! 4 x 3 means 4 + 4 + 4 (three groups of 4)."]
```

**Adaptation by grade:**
- Grade 1: Addition/subtraction to 20, no multi-digit section (replace with missing number problems)
- Grade 2: Addition/subtraction with regrouping
- Grade 3: Multiplication/division facts
- Grade 4: Multi-digit multiplication and long division
- Grade 5: Decimal operations, order of operations

### Template M2: Place Value / Number Sense

**Used for:** Counting, place value, comparing numbers, skip counting

**Structure:**
```
[Title: e.g., "Numbers to 120 & Place Value"]
[TEKS reference]
[Name/Date lines]

Section 1: "Fill in the missing numbers:" (2-3 number sequences with blanks)
- Sequences show a pattern; student fills gaps
- Include forward and backward counting
- Format: "38  ___  40  ___  42  ___  44  ___  46"

Section 2: "Write how many tens and ones. Then write the number."
- 6-8 visual representations (base-ten blocks described as groups)
- Format: "___ tens ___ ones = ___"

Section 3: "Compare: Write >, <, or ="
- 8 comparison pairs
- Mix of obvious and tricky (same digits reversed, numbers close together)
- Format: "45 O 54    78 O 78    31 O 29    100 O 99"

Section 4: "Skip count! Fill in the blanks:"
- 3 sequences (by 2s, 5s, 10s)
- Some numbers given, some blanked out
- Format: "Count by 2s:  2  ___  6  ___  10  ___  14  ___  18"

[Footer tip: "Place value: the tens place tells how many groups of 10. The ones place tells how many extras!"]
```

### Template M3: Story / Word Problems

**Used for:** All grades, applied math

**Structure:**
```
[Title: "Story Problems"]
[TEKS reference]
[Name/Date lines]
[Instruction: "Read carefully! Show your work. Write the answer in a sentence."]

8-10 story problems, each formatted as:

"#1 [Story problem text, 2-3 sentences max]

Equation: ________________________

Answer sentence: ________________________"
```

**Content rules for story problems:**
- Use realistic scenarios (NOT classroom settings)
- Good settings: parks, home, baking, shopping, sports, pets, family outings, building things
- Vary the operation required (don't make all addition or all multiplication)
- Include "tricky" problems that require interpretation (e.g., "How many vans are needed?" where remainder means you need one more)
- For Tier 3: weave the child's name into problems ("Noelle has 36 stickers...")
- Grade K-1: Include space for drawing a picture
- Grade 3-5: Include multi-step problems (2 operations needed)
- Include at least one comparison problem ("how many MORE does X have than Y?")

**Footer tip:** "Read the problem twice! Underline the important numbers. Ask: do I multiply or divide?"

### Template M4: Time (Telling Time / Elapsed Time)

**Used for:** Grades 1-3

**Structure:**
```
[Title: e.g., "Telling Time & Elapsed Time"]
[TEKS reference]
[Name/Date lines]

Section 1: "Write the time shown on each clock:" (6-12 clock faces)
- Clock faces described as: "Clock showing [hour hand position] and [minute hand position]"
- In PDF: rendered as simple circle with numbers 1-12 and two lines for hands
- Mix of on-the-hour, half-hour, quarter-hour, and 5-minute intervals (grade-appropriate)

Section 2: "Draw the hands on the clock to show the time:" (4 clocks, grade 1-2)
- Blank clock faces with the target time written below
- Format: "3:00    6:30    10:00    1:30"

Section 3 (Grade 3+): "Elapsed Time: How much time has passed?"
- 4-6 elapsed time problems
- Mix of: given start/end find elapsed, given start/elapsed find end, given end/elapsed find start
- Include real-world contexts ("Movie starts at 4:15 PM and is 1 hour 45 minutes long. What time does it end?")

[Footer tip: Grade 1: "The short hand points to the HOUR. The long hand points to 12 for o'clock or 6 for half past!" / Grade 3: "For elapsed time, try counting by hours first, then add the extra minutes!"]
```

### Template M5: Fractions

**Used for:** Grades 1-5 (complexity scales)

**Structure:**
```
[Title: "Fractions"]
[TEKS reference]
[Name/Date lines]

Section 1: Visual fractions
- Grade 1-2: "Are the parts EQUAL? Circle YES or NO" + "Shade the fraction: 1/2, 1/4, 2/4"
- Grade 3+: "What fraction is shaded? Write it on the line." (shapes divided into parts with some shaded)

Section 2: Number line fractions (Grade 3+)
- "Label the fractions on the number line:"
- Format: "0  ___  ___  ___  1" with denomination specified (e.g., "fourths")

Section 3: Comparing fractions (Grade 3+)
- "Compare: Write <, >, or ="
- 6 comparison pairs
- Mix: same numerator different denominator, same denominator different numerator, equivalent fractions

Section 4: Equivalent fractions (Grade 3+)
- "Fill in the missing number to make equivalent fractions:"
- Format: "1/2 = ___/4    1/3 = 2/___    2/4 = ___/2"

[Footer tip: "Equivalent fractions are different ways to name the same amount. 1/2 = 2/4 = 3/6!"]
```

## ELA PAGE TEMPLATES

### Template E1: Reading Comprehension - Fiction

**Structure:**
```
[Title: "Reading Comprehension: Fiction"]
[TEKS reference]
[Name/Date lines]

[ORIGINAL PASSAGE: 150-250 words]
- Grade 1-2: Simple narrative, familiar setting, clear problem/solution
- Grade 3-5: More complex narrative, character development, theme
- MUST be original (never copied from any source)
- Use vivid, age-appropriate language
- Include dialogue
- Have a clear beginning, middle, end

[5 multiple-choice questions, STAAR format:]
1. Main idea / what is the story mainly about
2. Character feelings / motivation
3. Why did [character] do [action]? (inference)
4. Vocabulary in context ("What does [word] most likely mean?")
5. Best evidence / which sentence best shows [claim]

Each question has 4 answer choices (A, B, C, D).
One correct, one close-but-wrong, two clearly wrong.

[Footer tip: "STAAR Tip: Go back to the story to find evidence for your answer!"]
```

**Passage quality rules:**
- Original characters with names
- Sensory details (not just plot summary)
- Age-appropriate conflict and resolution
- No violence, no scary content, no adult themes
- Alternate fiction and nonfiction across weeks
- Vary genres: realistic fiction, fantasy, historical fiction, adventure, mystery

### Template E3: Phonics / Word Study

**Used for:** Grades K-2 primarily, word study for 3-5

**Structure (Grades 1-2):**
```
[Title: "Phonics Review" or specific pattern like "Long Vowels"]
[TEKS reference]
[Name/Date lines]

Section 1: Fill in missing letters (10 words)
- CVC words with vowel blanked: "c_t, p_g, b_d"

Section 2: Word transformation (6 pairs)
- "Add silent E to change the word. Write the new word:"
- "cap + e = ___"

Section 3: Digraphs/blends (8-12 fill-in)
- "Fill in SH, CH, TH, or WH to finish the word:"
- "___ip, ___at, ___in"

Section 4: Beginning blends (8-10 fill-in)
- "Fill in the beginning blend:"
- "___og, ___um, ___ap"

Section 5: Vowel teams (6 words)
- "Circle the vowel team you hear:"
- "boat (oa / ou)  rain (ai / aw)  feet (ea / ee)"

[Footer tip: "When two vowels go walking, the first one does the talking! (ai, ea, oa, ee)"]
```

**Structure (Grades 3-5): Vocabulary & Word Study**
```
Section 1: Prefix/suffix meanings (8-10 words)
- Given prefix/suffix definitions, determine word meaning
- "un- = not, re- = again, pre- = before, -ful = full of, -less = without"
- "unhappy means: ___"

Section 2: Context clues (3-4 sentences)
- Bold word in sentence, multiple choice for meaning
- "The ENORMOUS elephant was so big it could barely fit through the gate."
- "Enormous most likely means: A. tiny  B. very large  C. fast  D. friendly"

Section 3: Multiple meaning words (3 sentences)
- Same word, different meanings based on context
- "I need to CHECK my answers. Check means: A. a pattern on fabric  B. to look over carefully  C. a type of payment"

[Footer tip: "Read the WHOLE sentence around an unknown word for clues to its meaning!"]
```

### Template E4: Grammar & Language

**Structure:**
```
[Title: "Grammar & Language Review" or specific topic]
[TEKS reference]
[Name/Date lines]

Section 1: Parts of speech identification
- Grade 1-2: "Sort each word: Write N (noun), V (verb), or A (adjective)" (15 words)
- Grade 3-5: "Write NOUN, VERB, ADJECTIVE, or ADVERB for each underlined word" (sentences with underlined words)

Section 2: Sentence-level practice
- Grade 1-2: "Underline the NOUN / Circle the VERB / Add an ADJECTIVE"
- Grade 3-5: "Add the correct punctuation" and "Rewrite with correct capitalization"

Section 3: Application
- Grade 1-2: "Add an adjective to finish each sentence"
- Grade 3-5: "Circle the correct word" (subject-verb agreement, pronoun usage)

[Footer tip: Grade 1: "A noun names a thing. A verb shows action. An adjective describes!" / Grade 3: "STAAR tests grammar IN CONTEXT -- practice finding errors in real sentences!"]
```

[Title: "Writing Skills Review"]
[TEKS reference]
[Name/Date lines]

### Template E7: Story Elements / Literary Elements

**Structure:**
```
[Title: "Story Elements & Comprehension" or "Text Features & Literary Elements"]
[TEKS reference]
[Name/Date lines]

Section 1 (Grade 1-2): Read a short passage, then answer:
- "Who are the CHARACTERS?"
- "What is the SETTING?"
- "What is the PROBLEM?"
- "How is the problem SOLVED?"
- "How does [character] feel at the beginning/end?"
- "What lesson does this story teach?"

- "Match each story element to its definition"
- Setting, Characters, Plot, Theme, Conflict

- "First person or third person? Write 1st or 3rd" (4 sentences)

Section 3: Text features (Grade 3-5)
- "What text feature would BEST help you with each task?"

Section 4: Genre identification (Grade 3-5)
- "What genre?" with 5 descriptions to classify

[Footer tip: "Characters = who. Setting = where & when. Problem = what goes wrong. Solution = how it's fixed!"]
```

## ANSWER KEY FORMAT

Answer keys go at the end of the packet, organized by page.

```
===ANSWER KEY===

[Page Title] Answers:
1. [answer]  2. [answer]  3. [answer]  4. [answer]
...

[Next Page Title] Answers:
1. [answer]  2. [answer]  ...
```

For story problems, include the equation AND the answer sentence.
For reading comprehension, just the letter (A, B, C, or D).
For open-ended writing sections, mark "Answers will vary" with a note on what to look for.

## ASHLEY'S TIP FORMAT (Page 1 Only)

Appears in a styled box at the top of the first page of each packet.

**Structure:**
```
Ashley's Note:
[2-3 sentences in Ashley's voice. Specific to this week's content.
Tells the parent what to focus on, what to skip if the kid is ahead,
what to spend extra time on if they're struggling. Honest, practical, warm.]
```

**Examples:**
- "This week we're hitting multiplication hard, specifically the 7s, 8s, and 9s. If your kiddo already flies through the speed round, skip ahead to the word problems; that's where the real thinking happens. Don't stress the timed element; speed comes with confidence."
- "Elapsed time is one of those topics that just takes a while to click (pun intended). If your child is frustrated, try acting it out with a real clock. Move the hands together and count. Way more effective than worksheets alone."
- "The reading passage this week is nonfiction about monarch butterflies. If your kid loves it, this is a great week to watch a documentary together and do a nature journal entry. The comprehension questions are STAAR-style, so practice going back to the text for evidence."
