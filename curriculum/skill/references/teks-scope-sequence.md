# TEKS Scope & Sequence: Weekly Pacing Guide

This file contains the week-by-week pacing for each grade level.
Each week lists the specific TEKS standards covered for Math and ELA.
The sequence ensures full standards coverage across 36 weeks with built-in review cycles.

## How to Use This File

When generating a packet for Week X, Grade Y:
1. Find the grade section below
2. Find the week number
3. Use the listed TEKS standards as your content targets
4. Cross-reference with `page-templates.md` for the page structure

## Notes on Pacing

- **Weeks are flexible.** Some families move faster or slower. The scope and sequence is a guide, not a mandate. Ashley's Tip should acknowledge this when relevant.
- **Spiral review is built in.** Weeks 31-34 are dedicated review for every grade. Additionally, story problems throughout the year spiral in earlier concepts.
- **STAAR prep intensifies in grades 3-5** (STAAR begins in 3rd grade). Weeks 31-36 include STAAR-format questions.
- **Summer packets** (optional future product): condensed 8-week review packets that prevent summer slide. Covers the most critical standards from the prior year.
